可以把属性看作一个对象的性质，把方法看作对象的动作，把事件看作对象的响应。

举个例子

比如有个视频播放器，它是一个对象，它会有那些属性呢方法事件呢？

属性:

它有属性width和height，使用这可以将它们当做一个变量来复制，从而调整它的尺寸。

有些属性只允许读取，不允许修改，比如视频播放器就应该有个属性叫duration，用来获取视频的总时间，显然是无法修改的。

方法:

视频播放器有个方法是play()，有个方法是pause()，用来播放和暂停

事件:

事件是从对象内部产生，并向外传播到开发者，以便能够执行事件过程中的代码。

例如视频播放器有个事件是ended，但视频播放完毕会触发这个事件。

开发者可以监听这个事件，然后在播放完毕后执行动作。

例子

获取属性

var myVideo=document.getElementById("myVideo");//获取对象
var a=myVideo.duration;
alert(a);

设置属性

var myVideo=document.getElementById("myVideo");
var a=parseInt(window.prompt("请输入宽度","600"));
myVideo.width=a;

方法调用

var myVideo=document.getElementById("myVideo");
myVideo.play();
var myVideo=document.getElementById("myVideo");
myVideo.pause();

事件响应

var myVideo=document.getElementById("myVideo");
myVideo.onended=function(){
	alert("播放完了");
};
